# Skybox Assets

Place the extracted `free_-_skybox_in_the_cloud/` folder in this directory to
use it as the Tellus skybox. If that model is not available, Tellus also tries
`free_-_skybox_in_the_cloud.glb`, `skybox_skydays_3.glb`, and then the bundled
basic skybox.

The app loads it from:

```text
/skybox/free_-_skybox_in_the_cloud/scene.gltf
```
